CalendarServiceGrpc

======================================================================================

CalendarServiceGrpc

---------------------------------------------------------------------------------------

gRPC service for the Instructor Calendar module. Manages physical class / event

scheduling: create, list, fetch by id, cancel, and reschedule calendar events.

Optional link to a course (assignedCourse). Identity/tenant from gRPC metadata.

======================================================================================

Method NameRequest TypeResponse TypeDescription
CreateCalendarEvent .CreateCalendarEventRequest .CreateCalendarEventResponse

Creates a new calendar event (physical class session).
FetchCalendarEvents .FetchCalendarEventsRequest .FetchCalendarEventsResponse

Fetches calendar events with optional filters (course, date range, type, status) and pagination.
FetchCalendarEventById .FetchCalendarEventByIdRequest .FetchCalendarEventByIdResponse

Fetches a single calendar event by ID.
CancelCalendarEvent .CancelCalendarEventRequest .CancelCalendarEventResponse

Cancels a calendar event (e.g. sets status to CANCELLED).
RescheduleCalendarEvent .RescheduleCalendarEventRequest .RescheduleCalendarEventResponse

Reschedules a calendar event to a new date/time.

LearnerCourseServiceGrpc

======================================================================================

LearnerCourseServiceGrpc

---------------------------------------------------------------------------------------

gRPC service definition for the Learner Course module in the afefe-learner microservice.

Provides endpoints for learners to browse, search, filter, and interact with courses,

manage wishlists, view course content, take quizzes, submit ratings, and manage notes.

User identity and tenant context are resolved from gRPC metadata (not request payloads).

======================================================================================
